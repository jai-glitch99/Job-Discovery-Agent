# AI Job Discovery Agent

This is an AI-powered Job Discovery Agent developed over 10-12 days to automate job searching and matching based on personal profiles, skills, and preferences.

## Phase 1: Project Setup and Basic Pipeline
The current iteration implements the basic pipeline, including uploading an Excel file containing user data/preferences and retrieving mock job data.

### How to Run
1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the Streamlit application:
```bash
streamlit run app.py
```
